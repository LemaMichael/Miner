## Requirements
- **macOS** 10.12+ 
- Swift 4.2+

## Install
Clone the repository and run <code>Pod install</code> using <a href="http://cocoapods.org">CocoaPods</a>\
**Or**\
Download latest release <a href="https://github.com/LemaMichael/Miner/releases/download/v1.0/Miner.zip">here</a>
